Downloads and Links.

Download

The addon is compatible with Blender version 2.8.0 and 2.7.9. The version for 2.7.9 is stripped down to import/export functionality.

  • For Blender version 2.8.0: rtcw_et_model_tools_bl280_v1.0.0_rc5
  • For Blender version 2.7.9: rtcw_et_model_tools_bl279_v1.0.0_rc2

Install instructions and more can be found in the User Guide or in the Tutorials Section.
